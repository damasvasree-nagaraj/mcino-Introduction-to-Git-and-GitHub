# Introduction to Git and GitHub

## Simple Interest Calculator

This project is part of the Coursera **Introduction to Git and GitHub** final assignment.

It contains a simple Bash script that calculates **simple interest** based on user input.

### Formula
Simple Interest = (Principal × Rate × Time) / 100

### Inputs
- **p** : Principal amount  
- **r** : Annual rate of interest  
- **t** : Time period in years  

### Output
- Calculated simple interest value

### Usage
Run the script and enter the required values when prompted.

```bash
bash simple-interest.sh
